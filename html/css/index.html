Les polices

Déclarer une nouvelle police

@font-face
{
    font-family: 'MaSuperPolice';
    src: url('MaSuperPolice.eot') format('eot'),
    url('MaSuperPolice.woff') format('woff'),
    url('MaSuperPolice.ttf') format('truetype'),
    url('MaSuperPolice.svg') format('svg');
}

Formats de fichiers de police à connaitre

  • .ttf : TrueType Font. Fonctionne sur IE9 et tous les autres navigateurs.
  • .eot : Embedded OpenType. Fonctionne sur Internet Explorer uniquement, toutes versions. Ce format est propriétaire, produit par Microsoft.
  • .otf : OpenType Font. Ne fonctionne pas sur Internet Explorer.
  • .svg : SVG Font. Le seul format reconnu sur les iPhone et iPad pour le moment.
  • .woff : Web Open Font Format. Nouveau format conçu pour le Web, qui fonctionne sur IE9 et tous les autres navigateurs.

Le pseudo-format :active

Il permet de donner une apparence différente au moment de l'activation. Il est souvent utilisé pour les liens, pour fournir un feedback à l'utilisateur lui indiquant que son action est bien prise en compte.

/* Flexbox */
#conteneur
{
    display: flex;
    justify-content: space-around;
    flex-wrap: wrap; /* si, en rétrécissant l'écran, les éléments ne rentrent plus en gardant leur taille minimale, alors ils passent sur plusieurs lignes */
}
.element
{
    min-width: 200px;
    height: 50px;
    margin: 5px;
    line-height: 50px;
    text-align: center;
    color: white;
    font-weight: bold;
}
#element1
{
    background-color: red;
    flex: 1;
}
#element2
{
    background-color: green;
    flex: 1;
}
#element3
{
    background-color: blue;
    flex: 1;
}

/* Boutons */
.bouton_cliquable
{
    background-color: #FFA500;
    border-radius: 30px;
    color: white;
    padding: 10px;
    margin-left: 10px;
    box-shadow: 3px 5px 3px black;
}
.bouton_cliquable:hover
{
    text-decoration: underline;
}
.bouton_cliquable:active
{
    box-shadow: 3px 5px 3px black inset;
}
