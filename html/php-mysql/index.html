Les balises

Comme le PHP s'insère au milieu du code HTML, il a besoin de ses propres balises.

Accompagné de l'instruction echo, on peut faire afficher du texte.

Cliquez pour afficher le code


    <?php echo "Premier code en PHP" ?>

Les variables

Les types de variables

  • string : Chaîne de caractères (texte)
  • int : Nombres entiers et entiers relatifs (positifs et négatifs)
  • float : Nombres décimaux (avec un point et non une virgule)
  • bool : Booléens (true or false)
  • NULL : rien (absence de données)

Utiliser une variable

On déclare d'abord le nom de la variable avant de lui donner une valeur :

Cliquez pour afficher le code


    <?php 
        $age_visiteur = 20; 
        $nom_visiteur = "Aurélien";
        $lunette = true;
    ?>

Ensuite on peut la faire s'afficher (avec concaténation) :

Cliquez pour afficher le code


    <?php 
        $age_visiteur = 20;
        echo 'le visiteur a' . $age_visiteur . 'ans';
    ?>

La structure if... else

On introduit la condition avec if, à la suite on ajoute entre parenthèses la condition en elle-même, et enfin on déclare les instructions.

Cliquez pour afficher le code


    <?php 
    $age_visiteur = 20;

    echo "Quel est ton age ?";
    if ($age_visiteur == 20)
    {
        echo " L'age parfait !";
    }
    elseif ($age_visiteur < 20)
    {
        echo " Tu es trop jeune";
    }
    else ($age_visiteur > 20)
    {
        echo " Tu es trop vieux";
    }
    ?>

Les conditions multiples

Mots-clés à connaitre
Mot-clé Symbole équivalent

Les mots-clés permettent de poser plusieurs conditions à la fois

Cliquez pour afficher le code


    <?php 
    $age_visiteur = 20;

    if ($age_visiteur >= 20 AND $langue == "français")

        echo "Welcome to my website !";

    {
        echo "Tu ne devrais pas être la !"
    }
    elseif ($age_visiteur < 20 AND $langue == "anglais")
    {
        echo "You shouldn't be here !"
    }
    ?>

Les conditions condensées (ternaires)

Fait deux actions sur une même ligne : Teste la valeur d'une variable dans une condition ET affecte une valeur à une variable selon que la condition soit vraie ou non.

Dans cet exemple, la condition modifie la valeur du booléen $majeur suivant la valeur de $age.

Cliquez pour afficher le code


    <?php 
    $age = 24;

    if ($age >= 18)
    {
        $majeur = true;
    }
    else
    {
        $majeur = false;
    }
    ?>

Les boucles

La boucle while

Similaire aux conditions, la boucle while permet néanmoins de répéter les instructions, tant que la condition est remplie.

On peut par exemple créer une boucle qui compte le nombre de lignes qu'on lui demande de créer :

Cliquez pour afficher le code


    <?php 
    $nombre_de_lignes = 1;

    while ($nombre_de_lignes <= 100)
    {
        echo 'Ceci est la ligne n°' . $nombre_de_lignes . '
';
        $nombre_de_lignes++; /* cette instruction permet d'augmenter le nombre de ligne de 1 à chaque nouvelle ligne */
    }
    ?>

La boucle for

3 éléments composent la condition de cette boucle.

  1. L'initialisation : c'est la valeur de départ de la variable.
  2. La condition : tant qu'elle est remplie la boucle est exécutée.
  3. L'incrémentation : permet de rajouter 1 à la variable, à chaque tour de boucle.

La boucle for est donc plus adaptée, lors du précédent exemple :

Cliquez pour afficher le code


    <?php 
    for ($nombre_de_lignes = 1; $nombre_de_lignes <= 100; $nombre_de_lignes++)
    {
        echo 'Ceci est la ligne n°' . $nombre_de_lignes . '
';
    }
    ?>

Les arrey

Ce sont des variables plus complexes, qui peuvent prendre plusieurs valeurs. Ces valeurs sont rangées selon des clés qui permettent d'afficher la valeur que l'on souhaite.

Cliquez pour afficher le code


    <?php 
    // On crée notre array $coordonnees
    $coordonnees = array (
    'prenom' => 'François',
    // clé   =>   valeur
    'nom' => 'Dupont',
    'adresse' => '3 Rue du Paradis',
    'ville' => 'Marseille');
    /* on peut tout mettre sur une même ligne mais pour une question de lisibilité il est plus facile de séparer chaque nouvelle clé et sa valeur */
    ?>

Ensuite il reste à afficher la variable avec la clé que l'on souhaite :

Cliquez pour afficher le code


    <?php 
    echo $coordonnees['clé']; // "clé" peut donc être remplacé par prénom, nom, adresse ou ville pour l'exemple précédent.
    ?>

Les fonctions

Les fonctions servent à automatiser une action et elles s'adaptent en fonction des informations qu'on leur donne.

Après avoir rentré les paramètres d'une fonction, on récupère sa valeur dans une autre variable si besoin.

Ici, on utilise la fonction strlen qui compte le nombre de caractères d'une phrase :

Cliquez pour afficher le code


    <?php 
        $phrase = "Je suis une phrase qui contient ";
        $suite = " caractères. Et si on me modifie, mon nombre de caractères se modifie aussi !";
        $longueur = strlen($phrase);
        $longueur2 = strlen($suite);
        echo $phrase;
        echo $longueur + $longueur2;
        echo $suite;
    ?>

Ce code fait apparaitre la phrase : Je suis une phrase qui contient 111 caractères. Et si on me modifie, mon nombre de caractères se modifie aussi !

Les variables superglobales

Ce sont des variables générées automatiquement par PHP.

Elles sont particulières car elles sont générées à chaque fois qu'une page est chargée. Ce sont des array commençant par un underscore et écrites en majuscules.

Les plus connues sont :

  • $_SERVER : Valeurs renvoyées par le serveur. $_SERVER['REMOTE_ADDR'] par exemple, donne l'adresse IP du client qui a demandé à voir la page.
  • $_SESSION : On y retrouve les variables de session. Ce sont celles qui restent stockées sur le serveur le temps de la présence d'un visiteur.
  • $_COOKIE : Contient les valeurs des cookies enregistrés sur l'ordinateur du visiteur. Cela permet de stocker des informations sur le visiteur, pour se souvenir de son nom par exemple.
  • $_GET : Contient les données envoyées en paramètre dans l'URL.
  • $_POST : Contient des informations envoyées via un formulaire.
  • $_FILES : Contient la liste des fichiers envoyés via le formulaire précédent.
